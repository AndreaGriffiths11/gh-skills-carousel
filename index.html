The missing piece was distribution

Copying files manually like it's 2012.

gh skill fixes that. No more hunting through GitHub repos, copying markdown files, or managing versions by hand.
03 / 07

How it works

Install skills from any GitHub repo:
gh skill install github/awesome-copilot documentation-writer
Keep everything updated:
gh skill update --all
Pin to tags or commits. Provenance travels with the file.
04 / 07

Supply chain hygiene

Skills aren't verified by GitHub and may contain prompt injections.

Use gh skill preview before installing. Same hygiene as npm packages, newer attack vectors.
