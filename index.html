GitHub just shipped skill management

Agent skills now install like npm packages.

Here's what changed.

What is gh skill

Package manager for agent skills.

Install, update, and publish skills across GitHub Copilot, Claude Code, Cursor, and Codex.

Requires GitHub CLI v2.90.0+

Here's what changed

Before: Copy skill files manually

Now: Install skills like packages

No more file copying like it's 2012.

In practice

gh skill install github/awesome-copilot documentation-writer

Pin to specific versions:

gh skill install github/awesome-copilot documentation-writer --pin v1.2.0

Why this matters

Agent skills are instructions that shape what AI does in your codebase.

Now they have proper versioning, provenance tracking, and integrity checks.

Supply chain security for AI agents.

The bigger picture

From: Sharing skills by copying files

To: Distributing skills with dependency management

Skills follow the open standard at agentskills.io

Try it now

gh extension upgrade --all

gh skill install github/awesome-copilot documentation-writer

gh skill update --all
